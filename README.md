# Nofeeai_Next-web-free
這是一個基於Chatgpt-next-web 開源的LLM合集，並且這個合集免費的，只供給個人使用不得用於商業目的
除插件工具外，與原始專案保持一致 ChatGPT-Next-Web 主要功能



# 模型種類
我們現在只支援以下模型免費使用
```
claude-2 (100K)
gpt-3.5-turbo-1106 (16K)
gemini-pro-vision
```
